### ☀️ Day 1: Views + URLs (Return simple text)

Right now, we are going to ignore the database, ignore HTML templates, and ignore forms. We are just going to make our Django app return a simple text message.

#### 🚶‍♂️ Step 1: Write a simple View
A View is just a Python function that takes a request from a user and returns a response. 

Open your existing file at `demo/views.py` and let's add a fresh, simple function at the very bottom:

```python
from django.http import HttpResponse

# Add this at the bottom of demo/views.py
def day1_test(request):
    return HttpResponse("Hello Sanket! This is Day 1.")
```

👉 **What this does:**
1. It creates a logic block called `day1_test`.
2. It takes a web `request`.
3. It replies matching exactly what we want using: `HttpResponse` (which just means "send back this text").

Your turn! Add that code to your `demo/views.py` file, save it.
